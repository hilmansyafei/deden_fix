Aplikasi sistem pakar berbasis Android ini dibuat untuk membantu pengguna / user ketika ingin melakukan diagnosis terhadap gangguan sistem pencernaan ketika mengalami gejala / keluhan yang dirasakan.
  Aplikasi sistem pakar ini membatasi hanya untuk mendiagnosis penyakit pada organ lambung.

Fitur Aplikasi

1. Mulai Diagnosis
2. Informasi 4 jenis penyakit pada lambung
3. Informasi lokasi rumah sakit di Depok-Jawa Barat
4. Informasi riwayat penyakit pengguna
5. Panduan penggunaan aplikasi
6. Profil dan tujuan pembuatan aplikasi
